### Package Description:
**TurboJS** is a powerful utility library that offers a collection of high-performance functions for array and object manipulation in JavaScript. It consolidates a variety of commonly used methods, providing an all-in-one solution for developers seeking efficiency and simplicity in their code.

**What is it?**

TurboJS is a comprehensive package that encapsulates functions for array and object operations. From basic tasks like cloning and filtering to more advanced operations like partial application and error handling, TurboJS has it all.

**Why do we need to use this package?**
- **Efficiency:** TurboJS is optimized for performance, ensuring your code runs faster and smoother.
- **Convenience:** It brings together a wide array of functions under one umbrella, reducing the need to import multiple utilities.
- **Consistency:** Provides a consistent API for working with arrays and objects, making your code more readable and maintainable.
- **Modern:** Leverages modern JavaScript features for cleaner and more robust implementations.

### Frequently Asked Questions:

**Q1: How do I install TurboJS?**
**A1:** You can install TurboJS via npm with the following command:
```bash
npm install turbojs
```

**Q2: How do I use TurboJS in my project?**
**A2:** After installing, you can import and use TurboJS like this:
```javascript
import Turbo from 'turbojs';

// Example usage
Turbo.turboCloneArray([...]);
Turbo.turboApply([...]);
```
